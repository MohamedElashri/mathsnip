# Intro
mathpix api version
only Mac support. 


# Install
``` bash
pip install mathsnip
```


# Usage
Must use root permission. Because it needs to catch the global key.
``` bash
sudo mathsnip -i <app_id> -k <app_key>
```

wait untill you see . it takes about 3 secs for cv to init.
